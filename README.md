# MetaGenomics Analysis for the SuPacow Project
This repository contains code and workflows used to analyze rumen metagenomic data as part of the SuPacow project. The analyses support the findings from the article “Metabolic capabilities of key rumen microbiota drive methane emissions in cattle” (mSystems, 2025).
Article link: https://journals.asm.org/doi/10.1128/msystems.00601-25
